# MonkeySee - PyTorch

Modified code from the original [MonkeySee repository](https://github.com/neuralcodinglab/MonkeySee). The original repository is a PyTorch implementation of the MonkeySee method introduced in [MonkeySee: Space-time-resolved reconstructions of natural images from macaque multi-unit activity](https://proceedings.neurips.cc/paper_files/paper/2024/hash/aa7eb65738b5bc71c81848fba9111c97-Abstract-Conference.html).
